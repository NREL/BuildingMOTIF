class LabelSet(token_classes: List[URIRef], labels: List[TokenizedLabel])[source]#

Represents a collection of TokenizedLabels that have the same set of classes among their tokens. This is used to speed up the matching process as we can figure out which labels are compatible with a template by looking at the classes, and then batch process all the labels with the same set of classes.
